XGRAPH 4.30 - April 30, 2014 - 2d Data Grapher
------

This package contains XGraph 4.30 executable compiled for Linux (64-bit OS).
The executable is under the "bin" directory, called "xgraph".
A test data file, "test_xy2.dat" is in the top directory.

Usage:

  Make an alias to "bin/xgraph":
  Example:
	alias xgraph=/home/bart/XGraph/bin/xgraph

  Plot data files.
  Example:
	xgraph test_xy.dat

  Alternatively, navigate to the "bin" directory in your file-browser, 
  and double-click "xgraph".  Then under the "File" menu, select 
  "Read New File" to pick your data file(s) to plot.

This version re-establishes the "-soc" option.

For web documentation and updates, surf to:   http://xgraph.org/
